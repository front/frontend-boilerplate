# Element
## Typograhphy
### Header
Headers `<h1>` to `<h6>` have a visual hierarchy from large to small. There is also helper classes to get the style of the individual headings `.h1` to `.h6`.

Headers is scaled down on smaller screens.

    @example
    h1 Header level 1
    h2 Header level 2
    h3 Header level 3
    h4 Header level 4
    h5 Header level 5
    h6 Header level 6
